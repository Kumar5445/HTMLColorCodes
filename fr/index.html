Quels sont les codes de couleur HTML?

Les codes de couleur sont les façons de représenter les couleurs que nous voyons tous les jours dans un format que l'ordinateur peut interpréter et d'affichage. Communément utilisé dans les sites Web et d'autres applications logicielles, il existe une variété de formats, y compris les codes HEX de couleurs, les valeurs RGB et HSL, et les noms de couleurs HTML, entre autres.

Codes de Couleur HEX

Les plus populaires sont les codes de couleur Hex; trois numéros octets hexadécimaux (ce qui signifie qu'ils se composent de six chiffres), chaque octet, ou une paire de caractères dans le code Hex, représentant l'intensité de rouge, vert et bleu dans la couleur, respectivement.

#XXXXXX

Hex code octet valeurs varient de 00, qui est la plus faible intensité d'une couleur, à FF qui représente l'intensité la plus élevée. La couleur blanche, par exemple, est faite par chacune des trois couleurs primaires à leur pleine intensité, ayant pour résultat le code de couleur Hex de #FFFFFF.

#FFFFFF

Noir, l'absence de toute couleur sur un écran, est tout le contraire, à chaque couleur affichée à leur plus bas intensité possible et un code de couleur Hex du # 000000.

#000000

Comprendre les bases de Hex notation de code de couleur, nous pouvons créer des couleurs en niveaux de gris très facilement, car ils se composent d'intensités égales de chaque couleur:

#454545

#999999

Les trois couleurs primaires, rouges, verts et bleus, sont faites en mélangeant l'intensité la plus élevée de la couleur désirée avec les intensités plus faibles des deux autres:

#FF0000

#00FF00

#0000FF

Avec les navigateurs modernes supportant le spectre complet des couleurs 24 bits, il y a 16 777 216 couleurs différentes possibles.
