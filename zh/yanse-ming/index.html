如何使用HTML颜色名

正如Hex颜色代码一样，HTML颜色名称可与内联HTML样式一同使用或在单独的CSS样式表中使用。 

HTML

<!-- In your HTML document -->
<body>
  <h1 style="color:IndianRed">Indian Red Title Text</h1>
  <p style="color:SlateGray">Slate gray paragraph text</p>
</body>

CSS

/* In your .css stylesheet */
h1 { color: IndianRed; }
p { color: SlateGray; }
